# Dotline — linear notes

A linear notebook: each note is a vertical line of connected points running down the center of the screen.

- **Free Dotline**: to the left of each dot — an editable alias (auto-numbered 1, 2, 3…, can be replaced with your own word); to the right — the main item name.
- **Reverse Dotline**: the first event is the anchor at the top. Each following event is added with an interval from the previous one. Time is shown on the left, event name on the right. The interval between dots is editable; changing it shifts all later events while the anchor stays fixed.
- Link labels sit in the gap between dots.
- Drag a dot sideways to delete it — the SVG string stretches, fades, and snaps.
- Sign in with Google to sync notes across devices.
